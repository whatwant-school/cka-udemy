### yaml
  - yml 파일 만들 때 아래와 같이...
```
$ kubectl run redis --image=redis123 --dry-run=client -o yaml > pod.yaml
```
  - yml 수정할 때 아래와 같이...
```
$ kubectl edit pod redis
```
